# BitzPrice

> Basic Next.js app that shows the current Bitcoin value in USD, GBP and EUR. Basic now but will build upon in the near future.

## Quick Start

``` bash
# Install dependencies
npm install

# Serve on localhost:3000
npm run dev

# Build for production
npm run build
```
